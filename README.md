# 루티니 (Routiny)

작은 루틴, 큰 변화 🌱

## 소개

루티니는 부담 없이 시작하는 습관 관리 앱입니다.
완벽하지 않아도 괜찮아요. 당신의 속도에 맞춰 천천히 함께 갑니다.

## 주요 기능

- ✅ **간편한 습관 등록** - 이모지와 색상으로 나만의 습관 꾸미기
- 📱 **홈 화면 위젯** - 앱을 열지 않아도 바로 체크
- 🌙 **오늘만 쉬기** - 컨디션이 안 좋은 날엔 하루 쉬어가기
- ⏸️ **일시정지** - 긴 휴식이 필요할 땐 잠시 멈추기
- 📅 **달력 보기** - 한눈에 보는 나의 습관 기록
- 📊 **통계** - 얼마나 꾸준히 해왔는지 확인
- 🔔 **따뜻한 리마인더** - 부드러운 메시지로 알림
