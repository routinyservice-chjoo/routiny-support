# Routiny
小さなルーティン、大きな変化 🌱

## 紹介
Routinyは気軽に始められる習慣管理アプリです。
完璧じゃなくても大丈夫。あなたのペースに合わせて、一緒にゆっくり進みましょう。

## 主な機能
- ✅ **かんたん習慣登録** - 絵文字とカラーで自分だけの習慣をカスタマイズ
- 🌱 **植物育成** - ルーティンを完了すると植物が育つ
- 📱 **ホーム画面ウィジェット** - アプリを開かずにチェック
- 🌙 **今日だけお休み** - 調子が悪い日は一日お休み
- ⏸️ **一時停止** - 長い休息が必要なときは一時停止
- 📅 **カレンダー表示** - 習慣の記録をひと目で確認
- 📊 **統計** - どれだけ続けてきたかを確認
- 🔔 **やさしいリマインダー** - 温かいメッセージでお知らせ
- 🏅 **コレクション** - 完成した植物を図鑑に登録
- 🎯 **最初の7日間ミッション** - 新規ユーザー向けの特別ミッションと報酬
